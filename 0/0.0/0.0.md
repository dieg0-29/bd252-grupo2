# 0.0. Antecedentes: Proyecto del curso de Modelado / Diseño

Elegimos el Sistema de Abastecimiento como la base de nuestro proyecto porque nos proporciona un punto de partida sólido y ya modelado para el desarrollo. Aunque el alcance final es más amplio e incluye módulos como Ventas, Inventario, Transporte y Clientes, el trabajo previo que hemos realizado en el área de abastecimiento nos da una idea de como abarcar el trabajo.

Este enfoque nos permite construir a partir de un cimiento ya establecido, ya que el proceso de abastecimiento es el que alimenta a toda la cadena de valor de la ferretería.
