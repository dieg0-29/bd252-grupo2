# Diseño Físico: **Módulo de Transporte**

## 𝄜 Tabla `DESPACHO`

| Año | Nuevos despachos | Total despachos | Crecimiento |
| --- | ---------------: | --------------: | ----------: |
| 1   |            3,600 |           3,600 |           — |
| 2   |            3,888 |           7,488 |          8% |
| 3   |            4,790 |          12,278 |         12% |

```ts
CREATE TABLE despacho (
  cod_despacho VARCHAR(100) NOT NULL,
  fecha_despacho DATE NOT NULL DEFAULT NOW(),
  hora_salida_despacho TIME NOT NULL DEFAULT NOW(),
  hora_regreso_despacho TIME NOT NULL DEFAULT NOW(),
  cod_estado_despacho INTEGER NOT NULL,
  cod_chofer VARCHAR(100) NOT NULL,
  cod_vehiculo VARCHAR(100) NOT NULL,
  PRIMARY KEY (cod_despacho),
  FOREIGN KEY (cod_estado_despacho) REFERENCES estado_despacho(cod_estado_despacho),
  FOREIGN KEY (cod_vehiculo) REFERENCES vehiculo(cod_vehiculo),
  FOREIGN KEY (cod_chofer) REFERENCES chofer(cod_chofer)
);
```

<br>

| Atributo              | Formato                       | Tamaño (bytes) |
| --------------------- | ----------------------------- | -------------: |
| cod_despacho          | **varchar(100)** *(prom. 20)* |             21 |
| fecha_despacho        | **date**                      |              4 |
| hora_salida_despacho  | **time**                      |              8 |
| hora_regreso_despacho | **time**                      |              8 |
| cod_estado_despacho   | **integer**                   |              4 |
| cod_chofer            | **varchar(100)** *(prom. 20)* |             21 |
| cod_vehiculo          | **varchar(100)** *(prom. 20)* |             21 |

**Tamaño de Fila: 87 bytes**

### **Parámetros físicos (POSTGRES):**<br>

```ts
CÁLCULOS  (FILLFACTOR = 80%)

Espacio útil por bloque = (8 192 − 24) × 0.8 = 6 534.4 bytes por bloque

Filas por bloque ≈ 6 534.4 / (87 + 23) ≈ 59.40 ≈ 59 filas por bloque

Bloques (a 3 años) ≈ 12 278 / 59 ≈ 208.09 ≈ 209 bloques

Tamaño total ≈ 209 × 8 192 = 1 712 128 bytes ≈ 1.63 mb
```

| Atributo                                             | Valor                                                           |
| ---------------------------------------------------- | --------------------------------------------------------------- |
| Tamaño de bloque                                     | 8 KB                                                            |
| Tamaño de extent                                     | —                                                               |
| Estructura del registro y tamaño del registro lógico | 110 bytes x fila *(87 + 23 de overhead)*<br>≈ 59 filas x bloque |
| Espacio libre reservado                              | `ALTER TABLE despacho SET (FILLFACTOR = 80);`                   |
| Tamaño total estimado (3 años)                       | **1.63 mb**                                                     |

## 𝄜 Tabla `DETALLE_DESPACHO_PARADA`

| Año | Nuevas paradas | Total paradas | Crecimiento |
| --- | -------------: | ------------: | ----------: |
| 1   |         18,000 |        18,000 |           — |
| 2   |         19,440 |        37,440 |          8% |
| 3   |         23,951 |        61,391 |         12% |

```ts
CREATE TABLE detalle_despacho_parada (
  cod_despacho VARCHAR(100) NOT NULL,
  cod_parada VARCHAR(100) NOT NULL,
  secuencia INTEGER NOT NULL,
  hora_salida TIME NOT NULL DEFAULT NOW(),
  hora_llegada TIME NOT NULL DEFAULT NOW(),
  cod_tipo_parada INTEGER NOT NULL,
  cod_estado_despacho_parada INTEGER NOT NULL,
  PRIMARY KEY (cod_despacho, cod_parada, secuencia),
  FOREIGN KEY (cod_estado_despacho_parada) REFERENCES estado_despacho_parada(cod_estado_despacho_parada),
  FOREIGN KEY (cod_tipo_parada) REFERENCES tipo_parada(cod_tipo_parada),
  FOREIGN KEY (cod_despacho) REFERENCES despacho(cod_despacho),
  FOREIGN KEY (cod_parada) REFERENCES parada(cod_parada)
);
```

<br>

| Atributo                   | Formato                       | Tamaño (bytes) |
| -------------------------- | ----------------------------- | -------------: |
| cod_despacho               | **varchar(100)** *(prom. 20)* |             21 |

| hora_salida                | **time**                      |              8 |
| hora_llegada               | **time**                      |              8 |
| cod_tipo_parada            | **integer**                   |              4 |
| cod_estado_despacho_parada | **integer**                   |              4 |

**Tamaño de Fila: 70 bytes**

### **Parámetros físicos (POSTGRES):**<br>

```ts
CÁLCULOS  (FILLFACTOR = 80%)

Tamaño total ≈ 877 × 8 192 = 7 184 384 bytes ≈ 6.85 mb
```
