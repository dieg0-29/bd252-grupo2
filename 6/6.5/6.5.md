# Tabla: `Ventas.producto`  *(250 filas)*

### Estructura lógica (por registro)

| Atributo        | Formato      | Tamaño                    |
| --------------- | ------------ | ------------------------- |
| cod_producto    | INT          | 4 bytes                   |
| nombre_producto | VARCHAR(120) | **≈ 40 bytes** (promedio) |

**Total (solo atributos):** **≈ 44 bytes x registro**
**Tamaño lógico con overhead:** 44 + 23 = **≈ 67 bytes x fila**

### Parámetros físicos (POSTGRES)

**CÁLCULOS:**

* Espacio Libre por Bloque = (8 192 − 24) × 0.8 = **6 534.4 bytes por bloque**
* Cantidad de Filas por Bloque = 6 534.4 / (44 + 23) = **≈ 97.53 ≈ 97 filas/bloque**
* Cantidad de Bloques = 250 / 97.53 = **≈ 2.56 → 3 bloques**
* Tamaño Total ≈ 3 × 8 192 = **24 576 bytes ≈ 0.023 MB**

### Parámetros

| Atributo                                             | Formato                                              |
| ---------------------------------------------------- | ---------------------------------------------------- |
| Tamaño de bloque                                     | 8 KB                                                 |
| Tamaño de extent                                     | —                                                    |
| Estructura del registro y tamaño del registro lógico | **≈ 67 bytes x fila** / **≈ 97 filas x bloque**      |
| Espacio libre reservado                              | `ALTER TABLE Ventas.producto SET (FILLFACTOR = 80);` |
| Tamaño total estimado                                | **≈ 0.023 MB**                                       |
