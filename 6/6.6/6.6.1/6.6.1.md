|1	|10000|	10000	|---|
|2	|10700|	20700	|7%|
|3	|11556|	32256	|8%|

```ts
CREATE TABLE MODULO_CLIENTES.CLIENTE
(
  cod_cliente UUID NOT NULL DEFAULT uuid_generate_v4(),
  fecha_registro_cliente TIMESTAMP NOT NULL DEFAULT NOW(),
  ultima_actividad_cliente TIMESTAMP NOT NULL DEFAULT NOW(),
  cod_persona UUID NOT NULL,

  FOREIGN KEY (cod_persona) REFERENCES MODULO_CLIENTES.PERSONA(cod_persona),
```

<br>

|Atributo|Formato|Tamaño|
|--------|-------|------|

|fecha_registro_cliente|**TIMESTAMP**|8 bytes|

**Total: 48 bytes x registro**

### **Parámetros físicos (POSTGRES):**<br>

```ts
CALCULOS: 

Espacio Libre por Bloque = (8 192 − 24) ∗ 0.8 = 6 534.4 bytes por bloque

Cantidad de Filas por Bloque = (6 534.4)/(48 + 23) ≈ 92.03 ≈ 92 filas por bloque

Cantidad de Bloques ≈ (32 256)/(92) ≈ 350.61 ≈ 351 bloques

Tamaño Total ≈ 351 ∗ 8 192 ≈ 2 875 392 bytes ≈ 2.75 mb
```

|Tamaño de bloque|8 KB|
|Tamaño de extent|---|
