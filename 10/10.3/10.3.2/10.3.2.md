# 10.3.2. Módulo 2 / Integrante 2

# Secuencias Σ

## Se crean secuencias para llevar un conteo de las operaciones clave del módulo de transporte.

### Contar número de Despachos Generados

Esta secuencia puede ser usada para llevar un conteo global de cuántas rutas de despacho se han creado en total, independientemente de los IDs de la tabla.

```
CREATE SEQUENCE "FERRETERIA".SEC_CONTADOR_DESPACHOS_GENERADOS
START WITH 1
INCREMENT BY 1
NO MAXVALUE;

SELECT NEXTVAL('"FERRETERIA".SEC_CONTADOR_DESPACHOS_GENERADOS');

SELECT * FROM "FERRETERIA".SEC_CONTADOR_DESPACHOS_GENERADOS;
```

### Contar número de Visitas (Paradas) Programadas

Similar a la anterior, esta secuencia puede usarse para contar el número total de paradas o entregas individuales que se han programado en el sistema.

```
CREATE SEQUENCE "FERRETERIA".SEC_CONTADOR_VISITAS_PROGRAMADAS
START WITH 1
INCREMENT BY 1
NO MAXVALUE;

SELECT NEXTVAL('"FERRETERIA".SEC_CONTADOR_VISITAS_PROGRAMADAS');

SELECT * FROM "FERRETERIA".SEC_CONTADOR_VISITAS_PROGRAMADAS;
```

### Contar número de Pedidos de Transporte (Contador Global)

Siguiendo el patrón de contadores globales, esta secuencia puede llevar un conteo total de cuántos `PEDIDO_TRANSPORTE` se han registrado en el sistema.

```
CREATE SEQUENCE "FERRETERIA".SEC_CONTADOR_PEDIDOS_TR
START WITH 1
INCREMENT BY 1
NO MAXVALUE;

SELECT NEXTVAL('"FERRETERIA".SEC_CONTADOR_PEDIDOS_TR');

SELECT * FROM "FERRETERIA".SEC_CONTADOR_PEDIDOS_TR;
```
