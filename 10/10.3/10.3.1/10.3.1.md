# 10.3.1. Módulo 1 / Integrante 1

# Secuencias ​Σ

### Contar numero de canjes

```sql
CREATE SEQUENCE MODULO_CLIENTES.SEC_CONTADOR_CANJES
START WITH 1
INCREMENT BY 1
NO MAXVALUE;

SELECT NEXTVAL('MODULO_CLIENTES.SEC_CONTADOR_CANJES');

SELECT * FROM MODULO_CLIENTES.SEC_CONTADOR_CANJES;
  ```
