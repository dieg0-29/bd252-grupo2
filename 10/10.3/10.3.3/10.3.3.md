# 10.3.3. Módulo 3 / Integrante 3

# Secuencias Σ

## Se crean secuencias para llevar un conteo global de las operaciones clave del módulo de almacén.

### 1. Contar número de Recepciones Procesadas

Esta secuencia puede usarse para llevar un conteo global de cuántas operaciones de recepción de mercancía (flujo R-302) se han completado en el almacén.

```
CREATE SEQUENCE "FERRETERIA".SEC_CONTADOR_RECEPCIONES
START WITH 1
INCREMENT BY 1
NO MAXVALUE;

SELECT NEXTVAL('"FERRETERIA".SEC_CONTADOR_RECEPCIONES');

SELECT * FROM "FERRETERIA".SEC_CONTADOR_RECEPCIONES;

```

### 2. Contar número de Conteos Cíclicos

Esta secuencia lleva un conteo total de cuántas tareas de conteo cíclico (flujo R-303) se han generado para auditar el inventario.

```
CREATE SEQUENCE "FERRETERIA".SEC_CONTADOR_CONTEOS
START WITH 1
INCREMENT BY 1
NO MAXVALUE;

SELECT NEXTVAL('"FERRETERIA".SEC_CONTADOR_CONTEOS');

SELECT * FROM "FERRETERIA".SEC_CONTADOR_CONTEOS;

```

### 3. Contar número de Incidencias Reportadas

Esta secuencia es un métrica clave para el Jefe de Almacén, contando todas las incidencias (productos dañados, faltantes, etc.) reportadas tanto en recepciones (R-302) como en conteos (R-303).

```
CREATE SEQUENCE "FERRETERIA".SEC_CONTADOR_INCIDENCIAS
START WITH 1
INCREMENT BY 1
NO MAXVALUE;

SELECT NEXTVAL('"FERRETERIA".SEC_CONTADOR_INCIDENCIAS');

SELECT * FROM "FERRETERIA".SEC_CONTADOR_INCIDENCIAS;

```
