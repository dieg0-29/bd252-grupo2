# 10.3.5. Módulo 5: Ventas

## Secuencias

---

- Contador de reclamos
```sql
CREATE SEQUENCE "FERRETERIA".sec_contador_reclamos
START WITH 1
INCREMENT BY 1
NO MAXVALUE;

SELECT NEXTVAL('"FERRETERIA".sec_contador_reclamos');

SELECT * FROM "FERRETERIA".sec_contador_reclamos;
```
---
- Contador de ventas
```sql
CREATE SEQUENCE "FERRETERIA".sec_contador_ventas
START WITH 1
INCREMENT BY 1
NO MAXVALUE;

SELECT NEXTVAL('"FERRETERIA".sec_contador_ventas');

SELECT * FROM "FERRETERIA".sec_contador_ventas;
```
